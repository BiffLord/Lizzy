# Lizzy
## Overview
This is puzzle game. The objective is to get
the water from the start to the end.
Along the way, there are colored obstacles,
some of which are open and some of which are closed.
Click the buttons to toggle the blocks of a specific color.
Grey is the walls and path, which are untoggelable, while
solid colors are closed blocks and outlines are open.
This game is based off the puzzle minigame from the
[Lazarus Pit](https://wynncraft.wiki.gg/wiki/Lazarus_Pit_(Quest))
quest of the MMORPG [Wynncraft](https://wynncraft.com/).

## Lizzy?
The game comes from Lazarus Pit. From there, it was
shortened to Lazzy. Then in homage to a great character
from a wonderful novel, and because Lazzy looks like
Lazy, it was renamed to Lizzy.
